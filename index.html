──────────────────────────────────────────────────────────
  what it does
──────────────────────────────────────────────────────────

  ▸ fully offline        runs on your machine, nothing leaves
  ▸ no filter            literally none
  ▸ fullscreen TUI       split pane, live stats, ascii doge
  ▸ tool use             reads files, writes files, runs commands
  ▸ teach mode           teach it stuff, it remembers across sessions
  ▸ model switching      swap models mid-session

──────────────────────────────────────────────────────────
  commands
──────────────────────────────────────────────────────────

  /teach <name>          start teaching it something
  /done                  save and exit teach mode
  /learnings             list saved learnings
  /model                 switch model
  /fans                  fan control settings
  /clear                 clear history
  /exit                  quit
